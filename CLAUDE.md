# 창업레이더

## 서비스 개요
서울시 예비 창업자를 위한 상권분석 + 금융지원 매칭 원스톱 웹 서비스
2026 서울시 빅데이터 활용 경진대회 창업 부문 출품작

## 핵심 기능 (우선순위 순)
1. 복수 입지 비교 — 두 지역을 나란히 선택해 매출·생존율·임대료 비교, AI 추천 이유 생성
2. 창업 시뮬레이터 — 자본금·업종·목표매출 입력 → 적합 상권 추천 + 예상 회수기간
3. AI 상담 챗봇 — OpenAI GPT-4o 기반, 상권 분석 + 금융상품 자연어 안내
4. 금융지원 매칭 — 소진공·서울신보·중진공 조건 매칭 → 신청 링크 연결

## 활용 공공데이터
- 서울 열린데이터광장: 서울시 생활인구, 상권분석 추정매출
- 공공데이터포털: 소상공인 상가정보
- 소진공: 서울시 기본 공개정보
- (필요 데이터 추가 예정)

## 기술 스택
- Next.js 14 App Router (TypeScript)
- Tailwind CSS
- OpenAI GPT-4o API
- 카카오맵 API
- Vercel 배포 예정 (변경 가능)

## 폴더 구조 규칙
- 페이지: /app 폴더
- 공통 컴포넌트: /components 폴더
- API 라우트: /app/api 폴더
- 타입 정의: /types 폴더
- 유틸 함수: /lib 폴더

## 개발 규칙
- 모든 UI 텍스트는 한국어
- 모바일 반응형 필수 (Tailwind 기준 sm/md/lg)
- 컴포넌트는 항상 TypeScript 타입 명시
- API 키는 절대 하드코딩 금지, 항상 .env.local 사용

## 차별점 (골목상권 서비스 대비)
- 복수 입지 동시 비교 기능
- 금융지원 조건 자동 매칭 + 신청 링크
- 자본금 기반 창업 시뮬레이션
- 생성형 AI 맞춤 추천문 생성
